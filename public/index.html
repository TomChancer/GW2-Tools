How it works

Combine 4 weapons of the same type in the Mystic Forge to receive a random exotic of that type.

Rare → Exotic rolls are the cheapest way to hunt precursors. Exotic → Exotic rerolls produce a higher base value output and can yield rare skins.

Notable targets

Rare skins (exotic rerolls)
⚔ Scythe — Staff exotics
🛡 Grinning Ghastly Shield — Shield exotics
Halloween event items — high value when available
Precursors (rare rolls)
Any weapon type can yield a precursor exotic. Drop rates are not published by ANet — each roll is independent.
